# Glossaire

## LLM

Large Language Model. Modèle de langage entraîné sur de grands corpus textuels.

## Agent

Système qui utilise un modèle pour décider d'actions, appeler des outils et produire un résultat.

## Tool

Fonction ou service externe qu'un agent peut appeler.

## Memory

Information conservée au-delà d'un simple message.

## State

État courant d'un workflow ou d'une exécution.

## Context

Informations effectivement envoyées au modèle à un instant donné.
