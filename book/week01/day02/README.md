# Day 2 — Neural Language Models and Embeddings

## Question centrale

Pourquoi a-t-on eu besoin de représenter les mots sous forme de vecteurs ?

## Objectif

Comprendre la transition entre les modèles statistiques classiques et les premiers modèles neuronaux du langage.

## Idée principale

Les LLM modernes reposent sur une idée fondamentale : transformer le langage en représentations numériques manipulables par des modèles.

Avant de comprendre les Transformers, il faut comprendre :

- pourquoi un mot ne peut pas rester une simple chaîne de caractères ;
- pourquoi le one-hot encoding ne suffit pas ;
- pourquoi les embeddings ont changé la manière de traiter le langage ;
- pourquoi les réseaux neuronaux ont permis de mieux apprendre les régularités du langage.
