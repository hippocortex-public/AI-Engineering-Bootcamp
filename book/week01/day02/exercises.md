# Exercices — Day 2

## 🟢 Exercice 1 — Compréhension

Expliquez avec vos mots la différence entre :

- one-hot encoding ;
- embedding.

Donnez un exemple de cas où le one-hot encoding échoue.

## 🟡 Exercice 2 — Similarité

Considérez les mots suivants :

```text
chat, chien, voiture, camion, pomme, banane
```

Regroupez-les par proximité sémantique.

Expliquez comment un bon espace d'embeddings devrait les organiser.

## 🟠 Exercice 3 — AI Engineering

Vous devez construire une recherche de documents pour une entreprise.

La recherche par mots-clés ne suffit plus.

Expliquez pourquoi une recherche vectorielle peut améliorer l'expérience utilisateur.

## 🔴 Exercice 4 — Architecture
