# Semaine 1 — Jour 7 — Premier client Python

## Objectif de la journée

Construire un premier client Python propre, testable et réutilisable pour appeler un LLM moderne.

Ce jour clôture la Semaine 1. Les jours précédents ont permis de comprendre l’histoire des modèles de langage, les embeddings, les Transformers, la tokenisation, le prompt engineering et les API modernes. Le Jour 7 transforme ces connaissances en un composant logiciel utilisable dans une application Python.

## Résultat attendu

À la fin de la journée, l’apprenant sait :

1. isoler la logique d’appel LLM dans un client dédié ;
2. gérer configuration, erreurs, timeouts et sorties structurées ;
3. écrire un mode mock pour tester sans dépendre d’un service externe ;
4. exposer une API Python simple pour une future application AI Engineering.
