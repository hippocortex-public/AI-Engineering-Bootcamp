# Références

## Documentation officielle

- OpenAI Python SDK — client applicatif et exemples d’instanciation.
- OpenAI Responses API — interface moderne de génération.
- Python `dataclasses` — structures typées simples.
- Python `json` — parsing et erreurs JSON.
- Pytest — tests unitaires.

## Concepts à retenir

- Un client LLM est une abstraction applicative.
- Un transport est une implémentation technique.
- Les sorties structurées doivent être validées.
- Les tests unitaires ne doivent pas dépendre du réseau.
- La configuration doit être externalisée.
