# Interview — Questions

## Questions conceptuelles

1. Pourquoi créer un client LLM interne au lieu d’appeler directement le SDK dans toute l’application ?
2. Quelle est la différence entre un client, un transport et une configuration ?
3. Pourquoi le mode mock est-il important en AI Engineering ?
4. Quels risques apparaissent quand un prompt est codé en dur dans une fonction métier ?
5. Pourquoi faut-il valider les sorties structurées d’un LLM ?
6. Quelle différence fais-tu entre une erreur de configuration et une erreur de validation ?
7. Pourquoi éviter de retourner uniquement une chaîne de caractères brute ?
8. Quels paramètres doivent être configurables dans un client LLM ?
9. Comment préparer un client pour un futur usage en production ?
10. Quand une abstraction de client est-elle excessive ?

1. Écris une signature Python pour une méthode `generate_text`.

3. Propose une structure de réponse typée.
4. Donne un exemple de test sans appel réseau.
5. Explique comment remplacer un transport mock par un transport réel.
