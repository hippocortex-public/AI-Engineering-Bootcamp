# Erreurs fréquentes

## Les LLM comprennent-ils le langage ?

Non. Ils modélisent une distribution de probabilité sur des séquences de tokens.

## Les systèmes à règles sont-ils obsolètes ?

Non. Ils restent excellents pour les domaines fermés, fortement réglementés ou critiques.

## Les LLM remplacent-ils toutes les autres approches ?

Non. Les systèmes modernes combinent recherche, règles, bases de données, outils et modèles.
