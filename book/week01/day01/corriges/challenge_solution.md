# Corrigés — Défis

Le choix dépend du besoin métier.

Une architecture réaliste combine souvent plusieurs approches :

- règles pour les cas critiques ;
- moteur de recherche pour les documents ;
- LLM pour le raisonnement et la génération ;
- outils pour agir ;
- mémoire pour personnaliser l'expérience.

La meilleure réponse est rarement « tout en LLM ».
