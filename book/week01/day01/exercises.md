# Exercices — Day 1

## Exercice 1 — Frise chronologique

Construisez une frise :

```text
ELIZA -> Systèmes experts -> n-grammes -> Réseaux neuronaux -> Transformers -> LLM
```

Pour chaque étape, indiquez :

- le problème résolu ;
- la limite restante ;
- le concept introduit.

## Exercice 2 — Choix d'architecture

Vous devez créer une FAQ interne qui répond à 20 questions fixes.

Choisiriez-vous :

- un système à règles ;
- un moteur de recherche ;
- un LLM ;
- un agent ?

Justifiez selon :

- coût ;
- maintenance ;
- latence ;
- complexité ;
- risque d'erreur.
