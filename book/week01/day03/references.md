# Références — Transformer et Attention

## Références principales

- Vaswani et al., *Attention Is All You Need*, 2017.
- Jay Alammar, *The Illustrated Transformer*.
- Stanford CS224N, lectures sur attention et Transformers.
- Documentation PyTorch, `torch.nn.MultiheadAttention`.
- Documentation NumPy, opérations matricielles.

## À retenir

Ces références sont utilisées pour consolider les concepts, mais le livrable du bootcamp reste autonome. L'objectif de cette journée est de comprendre la brique d'attention en tant qu'AI Engineer, pas de reproduire toute la littérature de recherche.

## Suggestions de lecture

1. Lire d'abord une explication visuelle du Transformer.
2. Relier ensuite les schémas au code NumPy du lab.
3. Revenir au papier original seulement après avoir exécuté l'attention from scratch.
